Silly Datatable

For angular 6+ versions apps. Easy styling datatable, all functionality is designed for use server side actions.

Get Silly Datatable

Capabilities

search

Quick search

The quick search component provides the flexibility to create a input textbox or use a external Angular form control.
find_in_page

Pagination

Pagination is a separate component with a pliable customization system.
settings

Options

Options allow to customize the display status of columns, and the number of items per page. User's data is stored for future use.
filter_list

Filters

Several types of filters are available in the filter component, and you can also create your own filters based on the available APIs.

Advantages

  • Server side actions for all features. There is nothing superfluous in component. Silly datatable has no logic.
  • All components are separated. You can use search, pagination, options, filters as nested in your app separately of datatable and set up appearance as you need.
  • Silly datatable has no hardcoded styles. Installed as empty table. Flexible for customization.
  • User's options will be saved for follow when user setup datatable. Used local storage.
