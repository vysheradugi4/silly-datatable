Filters settings

For setup appearance of filters component use settings:

<ngx-silly-datatable-filter [settings]="filterSettings"></ngx-silly-datatable-filter>

Object settings is a instance of class FilterSettings. It has the following properties for setup datatable filters component:

  • formContainerClass - Css class name for div container of filter.
  • cancelButtonTitle - Cancel button text ("Cancel" by default).
  • cancelButtonClass - Cancel button css class name.
  • submitButtonTitle - Submit button text ("Apply filters" by default).
  • submitButtonClass - Submit button css class name.
  • buttonsContainerClass - Css class name of the div container that contains buttons 'Cancel' and 'Apply filters'.
  • formElementsContainerClass - Class of the div container that contains the container with buttons and all form fields.

* - required property.

For example:

Don't forget import FilterSettings for init new instance:

import {{'{'}} FilterSettings {{'}'}} from 'silly-datatable';
