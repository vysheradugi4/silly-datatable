Installation

Integration in templates

For use Silly Datatable insert in your template component tag:

<ngx-silly-datatable></ngx-silly-datatable>

All features of Silly Datatable are separated components. This is made for easy integration into the user interface.

For add search field insert in template:

<ngx-silly-datatable-search></ngx-silly-datatable-search>

For add pagination feature insert in template:

<ngx-silly-datatable-paging></ngx-silly-datatable-paging>

For add options of table insert in template:

<ngx-silly-datatable-options></ngx-silly-datatable-options>

For add filters insert in template:

<ngx-silly-datatable-filter></ngx-silly-datatable-filter>

All components required additional setup. Please look details info in specific sections.
