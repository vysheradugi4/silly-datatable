Pagination settings

For setup appearance of pagination component use settings:

<ngx-silly-datatable-paging [settings]="paginationSettings"></ngx-silly-datatable-paging>
  • settings - Setting for customization appearance of pagination component (Instance of PaginationSettings class).

Don't forget import PaginationSettings:

import {{'{'}} PaginationSettings {{'}'}} from 'silly-datatable';

Pagination settings:

  • infoClass - Css class anme for customize information string "Page 1 of 10".
  • containerClass - Css class name for pagination div container.
  • arrowButtonClass - Css class name for buttons with arrows (start, end, next, prev).
  • numberButtonClass - Css class name for buttons with number of page.
  • start - "To Start" button word or symbol ("<<" by default).
  • prev - "Prev" button word or symbol ("<" by default).
  • next - "Next" button word or symbol (">" by default).
  • last - "To Last" button word or symbol (">>" by default).

* - required property.
