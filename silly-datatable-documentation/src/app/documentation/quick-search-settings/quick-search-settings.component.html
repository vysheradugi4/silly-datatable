• inputId - Id will be added to input element of search field.
  • searchInputClass - Css class name will be added to search <input> tag in search component.
  • prefix - Html template code for prefix before input form field (can use for label for example) (TemplateRef).
  • suffix - Html template code for suffix after input form field (can use as error notification for example) (TemplateRef).
  • disabled - Set input form field as disabled dynamically (boolean).
  • placeholder - For Define input form field placeholder.
  • dataFromExternalControl - Api for use other form control as search field.
  • usedExternalControl - The default search control will not be shown. Use dataFromExternalControl parameter for send search string in table params request (boolean).
