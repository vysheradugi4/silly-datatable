Row Clicked Event

For catch user's click or double click by row Silly Datatable provided output with event emitter.

Click:

<ngx-silly-datatable (rowClicked)="selected($event)"></ngx-silly-datatable>

Double click:

<ngx-silly-datatable (rowDoubleClicked)="dblClickSelected($event)"></ngx-silly-datatable>

After this need to create method-handler selected (for click) or dblClickSelected (for double click) in component.
