Form Fields

For configuration of filters component use form fields settings:

<ngx-silly-datatable-filter [formFields]="filterFormFields"></ngx-silly-datatable-filter>

Object formFields is array of instances of class FilterFormField. It has the following properties for configuration datatable filters component:

  • id - * Form field id.
  • name - * Form control name.
  • type - * Type of input. Available Values: 'textbox', 'dropbox', 'custom'.
  • value - Initial value of form field.
  • source - Accessory array of values for form control. (Can be used in selects etc.)
  • valueKeyName - Used in dropbox input, for define value key name for show data in select.
  • placeholder - Placeholder of form field.
  • controlClass - Form field css class name.
  • formGroupClass - Css class name for container that combined label and form field.
  • controlContainerClass - Div container of form field css class name.
  • labelContainerClass - Label container css class name.
  • formControlLabel - Title in label.
  • customInput - Custom input template (TemplateRef).
  • disabled - Disable form field.

* - required property.

Don't forget import FilterFormField for create new instance:

import {{'{'}} FilterFormField {{'}'}} from 'silly-datatable';
