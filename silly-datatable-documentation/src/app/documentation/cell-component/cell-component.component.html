Cell Component

When in cell placed custom component, Silly datatable provided to pass event from this component to output property.

<ngx-silly-datatable [settings]="settings" (componentCellEvent)="eventInCellComponent($event)"></ngx-silly-datatable>

Need to create method-handler eventInCellComponent for catch event.

For data exchange with cell component without additional services use componentCellEvent. For use event emitter in your component define public variable:

public componentCellEvent: EventEmitter<any> = new EventEmitter<any>();

For send data from cell component use call:

this.componentCellEvent.next(this.someData);

In public variable row in cell component will be able data of current row. Just add in component:

public row: any;
