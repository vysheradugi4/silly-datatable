Table Params

The tableParams object is is the instance of TableParams class. Contains current search string, sort params, filters params, pagination params and source for table.

<ngx-silly-datatable [(tableParams)]="tableParams"></ngx-silly-datatable>

This is two way binding property. In datatable parent component need to create private property, getter and setter for it.

For example:

Don't forget import class:

import {{'{'}} TableParams {{'}'}} from 'silly-datatable';

Table params contains:

  • searchText - Current search string from search component.
  • sort - Instance of Sort class. For data of sort by columns.
  • pagination - * Instance of Pagination class. Pagination data.
  • filters - Object, contains keys of filter and selected value.
  • source - Data source for placing data in a table (array).

* - required fields.
