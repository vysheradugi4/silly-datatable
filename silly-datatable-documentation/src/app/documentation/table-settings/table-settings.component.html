Main settings

For setup styles and data representation of Silly Datatable component use following attributes.

<ngx-silly-datatable [settings]="settings"></ngx-silly-datatable>

Object settings is a instance of class TableSettings. It has the following properties for setup datatable:

  • tableClass - Css class name will be added to <table> tag. Class for all table.
  • headerRowClass - Css class name for first row of table for header. Will be added in first <tr> tag.
  • rowClass - Css class name for each row in table except first (header) row. Will be added to each <tr> tag except first.
  • sortHeaderClass - Css class name for sortable header link. Will be added to <a> tag.
  • dataNotFoundPhrase - Text under heading when data is not found ('Data not found' by default).
  • itemsPerPageList - List of possible counts of items per page (hardcoded 10 by default). This is array of number. For example: [10, 30, 100].

* - required property.

For example:

Don't forget import TableSettings class:

import {{'{'}} TableSettings {{'}'}} from 'silly-datatable';
