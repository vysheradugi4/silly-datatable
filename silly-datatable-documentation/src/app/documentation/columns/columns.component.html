Columns

Object columns is a array instances of array of class Column. Column object has the following properties:

  • id - * Column id for link with source. Id string can be used as path for find value in object. For example: id: 'contact.address.city'
  • title - Name of column. Title used as text for header and sorting header of table.
  • headerClass - Custom css class for header of column.
  • cellClass - Custom css class for cells of current column.
  • sortable - Resolver for sorting current column (boolean).
  • show - Control for show or hide of the column (shown by default).
  • prepareCellFunction - Customization function for prepare data before showed.
  • componentCell - Custom component to display in the cell. Can be used for place buttons, links, form fields and etc. in cell. Property row in this custom component will be contains current source of row. Make sure that your component added to entryComponents section in your module. For data exchagange with componentCell use services or defined event emitter componentCellEvent. Details below.

* - required property.
