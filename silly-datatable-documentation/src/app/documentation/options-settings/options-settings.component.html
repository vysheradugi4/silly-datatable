Options component settings

For setup appearance of options component use settings:

<ngx-silly-datatable-options [settings]="optionsSettings"></ngx-silly-datatable-options>

Object optionsSettings is a instance of OptionsSettings class. It has the following properties for setup datatable options component:

  • containerClass - Css class name for options div container.
  • columnsSectionClass - Css class name for columns checkboxes section div container.
  • columnsTitleClass - Css class name for columns title section.
  • columnsTitle - Title text for sction of columns checkboxes (Columns: by default).
  • checkboxContainerClass - Css class name for containter of each checkbox with label.
  • checkboxClass - Css class name for checkbox.
  • labelCheckboxContainerClass - Css class name for containter of each checkbox label.
  • labelCheckboxClass - Css class name for checkbox label.
  • itemsPerPageSectionClass - Css class name for select count of pages section div container.
  • labelSelectClass - Css class name for select label.
  • itemsPerPageTitle - Title (label) text for select of count items per page (Items per page: by default).
  • selectClass - Items per page select's css class name.

* - required property.

Don't forget import OptionsSettings for create new instance:

import {{'{'}} OptionsSettings {{'}'}} from 'silly-datatable';
